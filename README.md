# Gushwork Assignment

## Project Overview
This repository contains a responsive static website built as an assignment for the Gushwork project. The site showcases a manufacturing process section with a modern stepper-style process flow, product details, image carousel, certifications, technical specifications, and several content sections designed for a manufacturing or industrial product landing page.

## Key Features
- Responsive landing page layout using HTML, CSS, and JavaScript
- Sticky announcement bar and responsive navigation menu
- Image carousel with thumbnail navigation and zoom functionality
- Manufacturing process section with a connected horizontal stepper
- Active step highlighting, progress line animation, and responsive scrolling
- Details section with certifications, product features, pricing, and CTA buttons
- Testimonials, portfolio, technical specifications, FAQ, and resource download sections
- Clean modern UI with consistent spacing, rounded cards, and minimal styling

## Manufacturing Stepper UI
- Displays 8 steps in a single row:
  - Raw Material, Extrusion, Cooling, Sizing, Quality Control, Marking, Cutting, Packaging
- Each step appears as a rounded pill button
- Steps are connected through a thin horizontal line running through the center
- Active step uses a solid primary dark blue background with white text
- Inactive steps use a light gray background with subtle border and dark text
- The progress line fills up to the active step for a polished stepper effect
- Fully responsive with horizontal scrolling on smaller screens

## Technologies Used
- HTML5
- CSS3
- JavaScript

## File Structure
- `index.html` — Main page markup and section structure
- `styles.css` — Layout, components, responsive design, and stepper styling
- `script.js` — Interactive behavior for tabs, carousel, menu, and progress updates
- `assests/` — Images, icons, and background assets used throughout the site

## Notes
- This project is a static site and can be opened directly in a browser
- The stepper component is implemented using CSS for styling and JavaScript for dynamic active state behavior
- Designed to match the requested manufacturing process stepper with a continuous connecting line
